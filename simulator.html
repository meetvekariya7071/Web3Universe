Block Simulator

Mine blocks, manipulate data, and watch the chain break — then fix it. Experience blockchain immutability firsthand.

🔢
SHA-256 Hash

A one-way cryptographic function — same input always gives the same output, but tiny changes completely alter the hash.

🎲
Nonce

"Number used once" — miners increment the nonce until the hash meets the difficulty target (starts with zeros).

⛓️
Immutability

Each block contains the previous block's hash. Changing any block invalidates all subsequent blocks in the chain.

Easy Hard 2 zeros

Target: Hash must start with 00
